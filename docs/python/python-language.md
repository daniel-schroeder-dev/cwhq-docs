### The `for` statement

A `for` statement is generally used to loop over a sequence, such as a `list`:


```python
fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)
```

Example output:
<hr>

```text
apple
banana
cherry
```
