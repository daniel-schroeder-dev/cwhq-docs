### add_image

```python
add_image(filename, size=None)
```

Adds an image to the page.

Parameters:

- <strong>filename</strong>(<em>str</em>): The filename.
- <strong>size</strong>(<em>int</em>): The size, in pixels (optional).

Returns:

- The image element.

Example usage:

<hr>
```python
taco_image = add_image("taco.png")
```
